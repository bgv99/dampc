### Library
Libraries can be included using the `include` keyword.

| Library | Description                         |
|---------|-------------------------------------|
| io      | I/O library.                        |
| time    | Time library.                       |
| math    | Mathematics library.                |
| data    | Library for common data structures. |
| net     | Network library.                    |
| mem     | Memory management library.          |
| rand    | Random generation library.          |
