### Types
| Type      | Size      | Example value | Description                           |
|-----------|-----------|---------------|---------------------------------------|
| `int`     | 4 bytes   | `123`         | Signed integer type.                  |
| `char`    | 1 byte    | `'a'`         | Unsigned ASCII character type.        |
| `boolean` | 1 byte    | `true`        | Boolean type.                         |
| `float`   | 4 bytes   | `1.23`        | Floating point type.                  |
| `double`  | 8 bytes   | `1.230000`    | Floating point type (high precision). |
